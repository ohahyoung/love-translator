# 💌 이 사랑 통역됩니다.

남녀(연인·부부) 언어 해석 서비스 데모.
텔레그램 대화 속 애매한 말을 **관계의 맥락**으로 통역해 줍니다.
외국어 번역기가 아니라, "됐어 오지마" 같은 말의 **속마음 후보 / 원하는 반응 / 위험한 답변 / 추천 답장**을 보여줍니다.

## 빠르게 보기 (데모 모드)

빌드 없이 바로 동작합니다.

```bash
# 방법 1: 그냥 파일 열기
open index.html        # macOS

# 방법 2: 정적 서버 (권장)
npx serve .            # 또는: python3 -m http.server 3000
```

`/api` 없이도 `presets.js` 프리셋으로 **모든 통역이 동작**합니다.

## 핵심 30초 데모 동선

1. **데모로 시작하기** → 사이/고민 선택 → **사랑통역봇 발급** → **연결된 방 보기**
2. 좌측 `🐰 우리 단둘이` 방 선택
3. 상대 메시지 **"됐어 오지마"** 의 `🔮 통역하기` 클릭
4. 우측에서 속마음 후보 / 위험한 답변(`알겠어. 안 갈게.`) / 추천 답장 확인
5. 추천 답장 클릭 → 입력창에 삽입 → 편집 → **전송**
6. (옵션) 우측 상단 **우리 커플 모드** 토글 ON → 이 커플 패턴 반영 해석으로 바뀜

다른 시나리오: `💍 신혼일기`의 "아무거나" / "나 괜찮아", `🌙 다툰 날`의 "먼저 자".

## Vercel 배포 + 실제 연동

```bash
vercel dev      # 로컬에서 /api 라우트까지 동작
vercel deploy   # 배포
```

서버리스 함수는 키가 있으면 실연동, 없으면 자동으로 프리셋 fallback 합니다.

| 환경변수 | 용도 | 없을 때 |
|---|---|---|
| `ANTHROPIC_API_KEY` | `/api/analyze` 가 Claude 로 해석 생성 | 프리셋/규칙 기반 해석으로 대체 |
| `TELEGRAM_BOT_TOKEN` | `/api/telegram` 가 실제 방/전송 처리 | 데모 방 목록으로 대체 |

> 온보딩 "실제 연동" 화면에서 토큰을 직접 입력해 `/api/telegram` 연결을 시도할 수도 있습니다(실패 시 데모로 안전 전환).

## 파일 구조

```
love-translator/
├── index.html        # 3패널 레이아웃 + 온보딩
├── styles.css        # 감성/실용 톤 디자인
├── app.js            # 온보딩·채팅·통역·전송 로직 (API 실패 시 프리셋 fallback)
├── presets.js        # 데모 방 + 통역 사전 (프론트/서버 공유, 시나리오 추가 지점)
├── api/
│   ├── analyze.js    # 메시지 해석 (Claude → 프리셋 fallback)
│   └── telegram.js   # 텔레그램 연동 골격 (getRooms / sendMessage)
├── vercel.json
└── README.md
```

## 새 시나리오 추가하기

`presets.js` 에서:
1. `rooms[].messages` 에 상대 메시지를 추가하고 `canTranslate: true` 표시
2. `interpretations["메시지 텍스트"]` 에 `general`(필수) / `couple`(선택) 해석 추가

## 안전 원칙

- 해석은 항상 **후보/가능성**으로 표현 (단정 금지)
- 상대를 조종하지 않고 **관계 회복형** 답장 추천
- 데모에는 가상의 대화만 사용
